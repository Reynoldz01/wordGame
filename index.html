Murdle

Inspiration from Wordle, but by me, Marcus Reynolds.
A site for my portfolio.
How to play:
- Start by guessing words, if a letter is in the word, it will show up underneath.
- The letters will appear in the order you guess them, once all five letters have been found, you will have to unscramble them to make the word you are looking for.

Ideas for Project

If i was to approach this project again, I would add several different things to improve it, such as:
- Display user guesses so they can keep track.
- Add CSS to the letter styling.
Ensure the submit text box has a character limit of 5, so people can't just spam the alphabet and encourage more thoughtful guesses.
- Mobile Responsiveness
- Colour Coding for when a correct letter is guessed in the correct place.
Increase word list from .... 3, by creating a seperate document/api link full of 5 letter words :)
